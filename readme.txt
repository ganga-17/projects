STEPS TO SETUP GIT AND THE GIT WORKFLOW:
   *First installed git
   *Then configure git
   *Then clone the remote repository to local repository
   *Initialize the directory for git project(git init)
   *add the files to staging area(git add .)
   *check the status of the git(git status)
   *Move files from staging area to git repository/create a commit(git commit -m "My First Commit")
   *push the code from local repository to remote repository(git push origin master)
   *Then create a file readme.txt in the remote repository
   *Then pull these changes to local repository
